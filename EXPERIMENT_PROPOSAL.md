**关键发现** (15,851 个样本):

1. ✓ Delayed ACK 样本确实系统性偏高 (平均 256ms)
2. ✓ 立即确认的数据段样本精度最高 (89.7% 准确)

## 三、关键技术改进

### 改进 1: 优化基线计算

**问题**: 当前使用 5% 分位作为基线，值过小 (0.064ms) 导致准确率低

**改进方案**:
```python
# 改用中位数或均值
baseline = np.median(baseline_values)
# 或使用绝对阈值
accuracy_threshold = 0.020  # 20ms
```

### 改进 2: 增强样本过滤

**当前逻辑**: MLT > 40ms 判定为 Delayed ACK

**改进方案**:
- 结合 RTT 估计动态调整阈值
- 考虑 TCP 选项中的时间戳
- 分析 ACK 间隔模式

### 改进 3: 补充统计检验

**新增内容**:
- t 检验：对比两组样本的均值差异
- KS 检验：对比分布形态差异
- 效应量计算：Cohen's d
